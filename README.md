Ndic
====

Naver English Dictionary on Command Line


## Installation

Either through forking or by using npm (the recommended way):

    npm install ndic -g

## Usage

    Usage: ndic <word>
    
    Example:
      $ ndic test  
      [명사](지식・능력 등을 알아보기 위한) 시험[테스트/검사]  
      [명사][동물] 겉껍질, 외각(shell); [식물] = TESTA
